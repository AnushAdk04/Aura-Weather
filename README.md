# Aura Weather App

Aura Weather is a modern weather web app that lets users search any city and view real-time weather conditions in a clean, responsive interface.

## Live Link

https://aura-weather-wine.vercel.app/

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- OpenWeather API
- Vercel (Hosting + Serverless Functions)

## Project Overview

The project uses a frontend weather dashboard with a serverless backend endpoint to fetch weather data securely. Users can search by city and view key details like temperature, humidity, wind speed, and weather conditions in an intuitive UI.
